Python print() function

The print() unction is used to print a specified message on the screen or other specified standard output device. The message can be string or any other object, the object will be converted to a string before written to the screen.

Calling print() function

You don't pass any arguments, but you will still need to put the empty parenthesis at the end, which tell Python to actually execute the function rather than just refer to it as a name
